**Brick Breaker Game**
**Overview**
The Brick Breaker Game is a classic arcade game developed using JavaFX. The objective of the game is to break all the bricks on the screen using a ball that bounces off a paddle controlled by the player.

**Features**
Multiple Levels: Progress through multiple levels with increasing difficulty.
Score Tracking: Keep track of your score as you break bricks.
Power-ups: Collect power-ups to enhance your gameplay experience.
Customizable Settings: Adjust game settings like ball speed and paddle size.
**Requirements**
Java Development Kit (JDK) 8 or higher
JavaFX SDK
